I built an open-source Unity plugin that integrates ElevenLabs AI voice generation directly into the Unity Editor.

**What it does:**
- Generate high-quality voiceovers without leaving Unity
- Batch process entire dialogue trees
- Auto-save as WAV assets or export as ZIP
- Access your generation history in-editor

This has been useful for rapid prototyping and creating placeholder voiceovers during development. The plugin handles the entire workflow from text input to usable AudioClip assets.

If you work with Unity and need voiceovers, it might save you some time.

GitHub: https://github.com/Yokesh-4040/ElevenLabs-Unity-Plugin
